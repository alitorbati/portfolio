Using node instructions from here: 
http://stackoverflow.com/questions/6084360/using-node-js-as-a-simple-web-server

// SERVER
To get it going, run the following command from the root directory
$ node server.js

// SASS
Sass needs to be running to compile css. Run the following from the stylesheets directory
$ sass --watch index.css.scss:index.css

// BROWSER
http://localhost:8080/tut/
